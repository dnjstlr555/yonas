플러그인 설명

  본 플러그인은 Berryz WebShare v0.952 (rev.1187)용 Album 플러그인을 모태로 하여 제작되었습니다.

  <중요> 프로그램 버전이 일치하지 않을 경우 정상 작동을 보증할 수 없으며, 이전 버전에 대한 하위 호환 계획이 없습니다.

  기본적으로 HTML5+CSS3 표준을 준수하여 Internet Explorer 8 ~ 11 및 크롬, 오페라, 파이어폭스 최신 버전에서 잘 작동합니다. (배포일 기준)

  IE 7 이하 버전이나 그 외의 웹브라우저에서의 정상 작동은 보증되지 않으며, 호환성 고려 대상에서 제외됩니다.

  본 플러그인은 반응형 CSS가 적용되어 해상도 기준으로 모바일과 데스크탑에 맞는 레이아웃으로 표시됩니다.

  IE8 이하에선 반응형 레이아웃이 적용되지 않으며, mp4/mp3 재생 기능을 이용할 수 없습니다.

  기타 사용상 버그나 호환성 문제는 배포자에게 문의하시기 바랍니다.



플러그인 사용상 참고 및 주의사항

  본 플러그인은 불특정 다수를 대상으로 한 불법자료 공유용이 아닌, 개인/소규모 사용을 염두하고 제작된 플러그인입니다.

  이미지 및 외부 라이브러리 사용을 최소화하여 의도된 디자인과 기능에 충실하게 코딩/최적화되었습니다.

  웹폰트, 채팅, SNS, 기타 외부 라이브러리 사용 및 소스 수정은 최종 사용자로서 본인 책임하에 실행해 주세요.

  개인적인 소스 수정 및 외부 라이브러리 사용에 의한 오류 문의는 받지 않습니다.

  본 플러그인 사용으로 발생하는 손해 또는 피해에 대해서, 제작자는 일절 책임 지지 않습니다.



본 플러그인에 사용된 라이브러리

  - jquery-1.12.4.min.js (jQuery 1.12.4 - IE 8 이하에서 사용됨)
  - jquery-3.2.1.min.js  (jQuery 3.2.1  - 기본값)



플러그인 요약

  - 윈도우 탐색기의 큰 아이콘 보기 스타일의 UI ("responsive v2.3 NE" 스킨 기반)

  - 우측 상단의 Detail view를 통해 목록 보기로 전환 가능 (모바일 뷰에선 팝업 메뉴)

  - 드래그/선택/우클릭 불가 (컨셉 + 오작동 방지(?) + 자체 우클릭 메뉴 사용)

  - 로그인/아웃 여부 판별 불가, 관리 메뉴 사용 불가 및 파일 정보 표시 불가 (본 플러그인 모듈의 한계)

  - 사진 파일 미리보기 가능

  - MP4/MP3 확장자에 대해 스트리밍 가능 (HTML5)

  - 항목 주소 복사 및 QR코드 표시 가능



적용 방법

  1. Berryz WebShare 실행 파일이 있는 폴더에 본 플러그인이 다음과 같은 구조가 되도록 압축 해제

     WebShare.exe
     Plugin
     └ IconView (본 플러그인 폴더명)
        └ IconView.dll 및 album_frame.html 등의 파일

  2. 프로그램 종료 및 재실행



본 플러그인 사용 시 Berryz WebShare 플러그인이 작동하지 않을 경우

  - Berryz WebShare 실행 파일이 있는 폴더에 플러그인 폴더/파일이 다음과 같은 구조로 존재하는지 확인

    WebShare.exe
    Plugin
    └ PlayList.asx
       └ playlist.asx.dll
    └ RARdownload
       └ RarDownload.dll
    └ ZIPdownload
       └ ZipDownload.dll

    (폴더명은 대소문자를 구분합니다.)
    (그 밖에 본 스킨에서 사용되지 않는 플러그인이 존재할 수 있습니다.)
